Projects

  • Digital Transformation at BGL

    BGL transitioned from its legacy web system, based on .NET 6, to a new API & Frontend architecture.

    For several years, I worked on connecting this API with the backend system. I wrote code that validated client requests and made changes to customer policies in our database.

  • Internal testing dashboard

    I wrote, from scratch, a new dashboard for displaying automated test results. This was displayed on TV screens in the office.

    The dashboard was a JSON API written in PHP, accessed by the frontend written in VueJS.

  • Supermarket Price Scanner

    I am currently working on a 'microsite' which will track prices of supermarket products over time, showing the increase (or decrease) in graphs for each product type.

    I have written the data-gathering script in Python. This script uses Playwright to load the supermarkets' websites, search for each product type, and then record the unit price, price per weight, and a dated screenshot of the page.

    The product search terms, and the results, are stored in a PostgreSQL database and the screenshots are stored in an AWS s3 bucket.

  • Tenant's Union website

    I wrote the website for the Peterborough Tenants Union, which includes the API integration with the payment processor to enable individuals to become members and pay their subs.

  • Tenant's Union design system

    I have been working on a design system for the Peterborough Tenants' Union website and online services. This is written with StencilJS in Typescript.

    The design system is based on the principles of the Gov.UK design system but with a bit more colour and flare.

    The repository is on my GitHub account.

  • Campaign data

    While I was in the Labour Party, I used the electoral data held by them along with geographical data from Ordnance Survey to produce graphical maps of voter locations and boundaries.
